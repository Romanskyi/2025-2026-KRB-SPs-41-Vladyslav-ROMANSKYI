# Автоматизація розгортання за допомогою GitHub Actions

GitHub Actions дозволяє автоматично збирати та публікувати вебпроєкт після внесення змін до репозиторію.

У межах дипломного проєкту ця технологія використовується для реалізації процесу безперервної інтеграції та доставки (CI/CD), що значно спрощує підтримку вебресурсу.
