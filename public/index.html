Автоматизоване розгортання та тестування вебпроєктів

Дипломний проєкт на базі Hugo, GitHub та GitHub Actions

Розроблений вебресурс демонструє реалізацію сучасного підходу до створення, автоматизованого тестування та безперервного розгортання статичних вебпроєктів із використанням технологій CI/CD.

Основні переваги

⚡ Автоматизація 🧪 Тестування ☁️ GitHub Actions 🚀 Hugo
Автоматичне оновлення після внесення змін Контроль якості перед публікацією Безперервна інтеграція Висока швидкість роботи

Про проєкт

Метою дипломної роботи є розробка сучасного вебресурсу університету із застосуванням автоматизованих засобів тестування та розгортання. Реалізований підхід дозволяє мінімізувати ручні операції, забезпечити стабільність роботи сайту та скоротити час публікації нових змін.

Використані технології

🚀 Hugo

Генератор статичних сайтів для швидкої та ефективної розробки вебресурсів.

🔄 Git

Система контролю версій для відстеження змін та підтримки командної розробки.

💻 GitHub

Платформа для зберігання вихідного коду та керування репозиторієм проєкту.

⚙️ GitHub Actions

Автоматизація тестування, збірки та розгортання вебпроєкту відповідно до концепції CI/CD.
